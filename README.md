# 她执导 · She Directs

华语女导演数据可视化项目。以数据驱动的方式，呈现 2000–2026 年华语女性导演的创作图景。

## 项目结构

```
├── index.html          # 主页面入口
├── css/                # 样式文件
├── js/                 # 交互逻辑与图表渲染
│   ├── chapters/       # 四大章节模块
│   ├── charts/         # ECharts / G6 图表组件
│   └── ...
├── assets/             # 图片、字体、音频等静态资源
├── data/               # 前端用 JSON/CSV 数据
├── 图表数据整理/        # 原始数据及处理中间文件
└── 特效/               # 词云、情感河流等独立页面
```

## 章节概览

| 章节 | 标题 | 内容 |
|------|------|------|
| 第一章 | 她们在哪儿 | 女导演年度占比、代际分布、出生地、存活曲线、入行路径 |
| 第二章 | 她写她，拍她 | 类型蝴蝶图、番位堆积条、女主角年龄对照 |
| 第三章 | 她说了什么 | 票房×评分象限、短评词云、议题网络图、情感河流 |
| 第四章 | 她们被看见了吗 | 电影节时间轴、三金漏斗、全球纬线、话语权综合指数 |

## 技术栈

- **ECharts** — 统计图表与地图
- **G6** — 力导向网络图
- **D3.js** — 散点图、词云布局
- **IntersectionObserver** — 视口懒加载
- 原生 JavaScript，无框架依赖

## 浏览方式

直接在浏览器中打开 `index.html`，或通过任意静态服务器访问：

```bash
npx serve .
```

## 数据来源

项目数据整理自公开的影视数据库、电影节官方记录及学术研究资料，涵盖 50 位华语女导演及其代表作品。
